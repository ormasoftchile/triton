Timeline Compiler — Example Gallery

Twelve timeline IR documents rendered with the Horizontal layout family, plus four Sequence Grammar examples (UML default and ByteByteGo themes), one Flow Grammar example (RAG pipeline, layered L→R layout), three Tree Grammar examples (default light theme and dark-teal theme — same IR, two themes), and one Composition Layer example (multi-panel poster assembling flow, tree, sequence, and stat grammars into a single infographic). Most timeline renders use the Consulting (Tier-1) theme, with a feature-rich Product showcase. Each example exercises a distinct combination of axis unit, track density, activity statuses, milestones, and edge-case behaviours. Use this contact sheet to bulk-check Phase 1–5 visual output.
