### Makefile Pipeline

```makefile
FIGURE_SRCS := $(wildcard figures/src/*.mmd)
FIGURE_PNGS := $(patsubst figures/src/%.mmd, figures/%.png, $(FIGURE_SRCS))

figures/%.png: figures/src/%.mmd
	timeline render $< --format png --scale 3 -o $@

pdf: $(FIGURE_PNGS)
	pdflatex ...
```

- Guard: if `packages/cli/dist/index.js` missing, builds core + cli first
- Deterministic: same source → same PNG bytes (Resvg is deterministic)
