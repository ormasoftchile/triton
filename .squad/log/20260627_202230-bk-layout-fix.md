### 2. Class Diagram Routing Fixes (Brian + Coordinator)

**Location:** `src/diagrams/class/layout.ts`

Three routing bugs fixed:

#### Bug 1: Port Ordering (Cascade Minimization)
- **Problem:** Naive even-distribution `t = (idx+1)/(n+1)` caused edge crowding and crossing
- **Fix:** Sort edges by opposite-end node center (Part 1 — crossing minimization); project onto wall with cascade algorithm (Part 2 — position enforcement)
- **Details:** 
  - MIN_PORT_GAP = 20px between adjacent ports
  - WALL_MARGIN = 16px inset from each wall end
  - Fallback to even distribution when space insufficient

#### Bug 2: Port GAP Enforcement
- **Problem:** Ports arriving at same wall were not properly spaced
- **Fix:** Cascade algorithm iteratively enforces MIN_PORT_GAP across forward/backward sweeps

#### Bug 3: Departure Point Targeting
- **Problem:** Departure ports aimed at node center, creating unnecessary diagonal paths
- **Fix:** Changed fallback from `borderPoint(..., bc.x, bc.y)` to `borderPoint(..., toPt.x, toPt.y)` so departure ports target actual arrival ports
